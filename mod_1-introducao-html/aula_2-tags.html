Tag <h1> até <h6>: Tags usadas para definir cabeçalhos e subcabeçalhos em uma página da web

Tag <p>: Tag usada para definir um parágrafo de texto

Tag <em>: Tag usada para aplicar enfâse (itálico) ao texto - possui semântica

Tag <i>: Tag usada para estilizar o texto, tipicamente exibindo-o em itálico - sem semântica

Tag <strong>: Tag usada para dar ênfase ao texto, geralmente exibindo-o em negrito - possui semântica

Tag <b>: Tag usada para aplicar estilo de negrito ao texto, mas ela é puramente estilística - sem semântica

Tag <img>: Tag usada para exibir imagens em uma página da web

Tag <a>: Tag usada para criar hyperlinks, que são links que permitem a navegação entre páginas da web, seções da mesma página, ou até mesmo para endereços de e-mail e outros recursos

Tag <video>: Tag usada para incorporar vídeos em uma página da web

Tag <audio>: Tag usada para incorporar conteúdo de áudio em uma página da web

Tag <label>: Tag usada para definir rótulos para elementos de formulário

Tag <input>: Tag usada para criar campos interativos em formulários
