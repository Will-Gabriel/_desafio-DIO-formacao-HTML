Breve explicação sobre as Tags Estruturais

  • <header>:

    • O que é: Representa a seção de cabeçalho de uma página ou de uma seção específica.
    • Para que serve: Normalmente contém o título, logotipo, menu de navegação ou outros elementos de introdução.

  • <nav>:

    • O que é: Indica uma seção da página destinada à navegação.
    • Para que serve: Contém links para outras partes do site ou para outras páginas. É usado para menus de navegação.

  • <main>:

    • O que é: Define o conteúdo principal da página, que deve ser único e relacionado diretamente ao objetivo da página.
    • Para que serve: Envolve o conteúdo central e essencial da página, excluindo cabeçalhos, rodapés, navegação, barras laterais, etc.

  • <section>:

    • O que é: Representa uma seção genérica de conteúdo, agrupando conteúdos relacionados.
    • Para que serve: Usado para dividir o conteúdo em blocos temáticos, como capítulos, partes de uma história ou seções de um artigo.

  • <article>:

    • O que é: Define um conteúdo independente e auto-suficiente.
    • Para que serve: Usado para artigos, posts de blog, entradas de fóruns, comentários, ou qualquer outro conteúdo que possa ser distribuído de forma independente.

  • <aside>:

    • O que é: Representa uma seção de conteúdo tangencialmente relacionado ao conteúdo ao redor.
    • Para que serve: Usado para barras laterais, notas de rodapé, ou qualquer conteúdo complementar que não faça parte do fluxo principal.

  • <footer>:

    • O que é: Representa o rodapé de uma página ou seção.
    • Para que serve: Contém informações como créditos, links para políticas de privacidade, ou qualquer informação de rodapé. Pode ser aplicado ao rodapé de uma página inteira ou de uma seção específica.
