Breve explicação sobre outras Tags muito utilizadas

  • <blockquote>:

    • O que é: Um elemento semântico que define uma citação longa ou em bloco, geralmente de uma fonte externa.
    • Para que serve: Envolve trechos de texto que estão sendo citados de outra fonte. O conteúdo é geralmente indentado ou estilizado para indicar que é uma citação. Pode opcionalmente incluir o atributo cite para indicar a origem da citação.
                          <blockquote cite="http://exemplo.com/fonte">
                        "Este é um exemplo de citação longa retirada de uma fonte externa."
                      </blockquote>

  • <q>:

    • O que é: Um elemento semântico que define uma citação curta ou inline.
    • Para que serve: Usado para incluir citações curtas dentro do fluxo de texto. O conteúdo dentro de <q> é geralmente renderizado com aspas (aspas duplas ou simples, dependendo da configuração do idioma).
                          <p>"Este é um exemplo de <q>citação curta</q>."</p>

  • <figure>:

    • O que é: Um elemento semântico usado para agrupar conteúdo independente, como imagens, diagramas, gráficos, códigos, ou ilustrações.
    • Para que serve: Serve para associar uma mídia (imagem, gráfico, etc.) com uma legenda opcional. Geralmente é usado para conteúdo que pode ser referenciado no texto, mas também pode ser separado do texto principal sem perder o sentido.
                            <figure>
                            <img src="imagem.jpg" alt="Descrição da imagem">
                            <figcaption>Legenda da imagem</figcaption>
                        </figure>

  • <figcaption>:

    • O que é: Um elemento semântico que define a legenda ou descrição de um conteúdo dentro de <figure>
    • Para que serve: É usado para fornecer uma descrição ou explicação do conteúdo contido em <figure>. Deve ser inserido dentro do elemento <figure>, normalmente antes ou depois do conteúdo da mídia.
                            <figure>
                            <img src="diagrama.jpg" alt="Diagrama de exemplo"/>
                            <figcaption>Este é o diagrama que ilustra o processo.</figcaption>
                        <figure>

  • <picture>:

    • O que é: Um elemento que oferece uma maneira avançada de trabalhar com imagens responsivas e de diferentes formatos.
    • Para que serve: Permite definir várias versões de uma imagem para serem carregadas dependendo de certos critérios, como a largura da tela (responsividade) ou o tipo de imagem suportado pelo navegador. Dentro de <picture>, usa-se o <source> para definir diferentes imagens e o <img> como fallback.
                            <picture>
                            <source media="(min-width: 800px)" srcset="imagem-grande.jpg">
                            <source media="(min-width: 400px)" srcset="imagem-media.jpg">
                            <img src="imagem-pequena.jpg" alt="Imagem responsiva">
                        </picture>
