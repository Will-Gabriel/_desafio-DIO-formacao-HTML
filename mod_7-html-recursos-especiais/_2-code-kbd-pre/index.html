Entendendo as Tags Code, Kbd e Pre

Tag Code

Utilizada para representar fragmentos de código em linha. O texto dentro desta tag geralmente é exibido em uma fonte monoespaçada, mas não aplica nenhum estilo adicional como espaçamento.

                
// Exemplo de uso:

                Use a tag <code><div></code> para criar uma divisão.

Resultado: Use a tag <div> para criar uma divisão.

Tag KBD

Representa uma entrada de teclado, como uma combinação de teclas que o usuário deve pressionar. O texto dentro desta tag também é exibido em uma fonte monoespaçada.

                
// Exemplo de uso:

                Para copiar esta informação, pressione <kbd>CTRL</kbd> + <kbd>C</kbd>

Resultado: Para copiar esta informação, pressione CTRL + C

Tag Pre

Usada para exibir texto pré-formatado. O conteúdo dentro da tag <pre> mantém os espaços em branco e quebras de linha, sendo exibido em uma fonte monoespaçada. É útil para exibir blocos de código ou texto que requerem uma formatação específica.

                
// Exemplo de uso:

                <pre>
                    .minha-div {
                    background-color: #000000;
                    color: #ffffff;
                    padding: 2px 4px;
                    border-radius: 2px;
                    }
                </pre>
                
                
// Exemplo de uso com code:

                <pre>
                    <code>
                    .minha-div {
                        background-color: #000000;
                        color: #ffffff;
                        padding: 2px 4px;
                        border-radius: 2px;
                    }
                    </code>
                </pre>
