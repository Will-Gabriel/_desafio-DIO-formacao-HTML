Entendendo as Tags Detais e Sumary

Tag details

Esta tag cria um contêiner que pode ser expandido ou recolhido pelo usuário. Dentro dela, pode haver qualquer conteúdo, como texto, imagens ou outros elementos HTML. Quando o contêiner está recolhido, apenas o conteúdo dentro da tag <summary> (se presente) é visível.

Tag sumary

Esta tag é opcional e usada dentro do <details>. Ela define o título ou cabeçalho que será mostrado quando o <details> estiver recolhido. Ao clicar no <summary>, o conteúdo adicional dentro do <details> é exibido.

                
// Exemplo de uso:

                <details>
                    <summary>Como trocar a cor de fundo de um elemento no HTML?</summary>
                    <p>Para executar esta ação, utlize o comando background-color no CSS</p>
                </details>

Resultado:

Como trocar a cor de fundo de um elemento no HTML?

Para executar tal ação, utilizamos o comando background-color no CSS.
